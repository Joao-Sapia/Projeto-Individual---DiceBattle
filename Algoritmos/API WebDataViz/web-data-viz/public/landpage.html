O que é DiceBattle?

RPG significa Role Playing Gaming, um jogo onde você cria um personagem com base no sistema de D&D, define os status, habilidades e história de um personagem. Depois disso você é livre para interpretar o seu personagem junto de um Mestre que é quem narra o mundo e define a história que acontece ao seu redor.
